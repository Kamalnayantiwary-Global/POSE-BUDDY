### 🌟 Overview
**Pose Buddy** is a sophisticated yet lightweight real-time photography assistant. It solves the common "pose-awkwardness" by providing a **Dynamic Ghost Overlay**. Using Computer Vision, it suggests the perfect posture based on your current environment—whether you are at a park, a graduation ceremony, or a city street.

> **"Don't just take a picture, frame a masterpiece."**

---

### 🔥 Key Features

* 🤖 **AI-Driven Suggestions:** Intelligent situational awareness that switches pose templates based on user input.
* 👻 **Ghost Silhouette Overlay:** Translucent, non-intrusive white outlines to guide your body alignment perfectly.
* ⚡ **Ultra-Low Latency:** Engineered with high-performance OpenCV backends for lag-free real-time processing.
* 📱 **Edge-Ready:** Optimized to run on standard hardware without the need for expensive GPUs.
* 🎞️ **Scanning HUD:** Built-in futuristic scanning animation for a professional "Terminator-style" UI/UX.

### 🕹️ Controls (Interactive Simulation)

Once the app is running, use these keys to simulate the AI's situational awareness:

- `1` : **Park Mode** (Suggests sitting/bench poses)
- `2` : **Graduation Mode** (Suggests celebratory/cap-tossing poses)
- `3` : **Street Mode** (Suggests standard portrait poses)
- `q` : **Quit Safely**
